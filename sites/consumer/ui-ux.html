UI / UX

User Interface & User Experience

User Experience Designers are concerned with how a user will interact and flow through a website.
User Interface Designers focus on the visual style and language of the
page. Often the lines between these two disciplines is blurred
and one person will perform both roles.

I had in my head what a UX designer was based on what I knew/thought I knew about a UI designer. When I think of UI, it think of the words it represents: User Interface. User Interface to me, meant the visual design of a web page/web app. So the UI designer would say create the all of the visual elements of a website and the layout. So what is left for the UX designer to do? From what I knew, I didn't have a clue. From what I'm reading, UI and UX are not the same, but they have some overlap in their roles.

UX design - considerations

  • social - you need to talk to people to get an idea of requirements
  • directing the behavior of the user
  • basics of design, layout, typography, visual communication
  • psychology
  • usability
  • what platform are we developing for? limitations?
