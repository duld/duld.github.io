Understanding Customer lifetime values (CLV)

  • Two main values of e commerce
  • Acquisition cost (AC or CoA)
  • the investment in time and money, needed to get someone to buy a product
  • CoA = cost per visit / conversion rate.
  • Customer lifetime values (CLV)
  • How much profit you will make from each customer?
  • The goal is to have a lower CoA than CLV. ie low expenses, high profits.
  • Ways to increase your CLV
  • Sell bundles
  • upsell add-ons
  • blackmail.

Securing Your Site

Understanding the basics of Secure Sockets Layer (SSL)

  • Security is important for three big reasons:
  • Your business could be financially liable for stolen credit cards in case of breach
  • You’ll lose customer confidence.
  • A site that looks secure increases conversions.
  • Unsecured traffic can be intercepted.
  • All sensitive information should be submitted on a secure page
  • HTTPS: Hypertext Transfer Protocol Secure
  • SSL: Secure Sockets Layer
  • You need an SSL certificate for your site to use HTTPS.
  • Many hosts will install an SSL certificate for you for a cost
  • PCI DSS Compliance: Payment card industry data security standard for handling credit card numbers.
  • PCI compliance requires software and hardware

Protecting Sensitive data

  • Just because you have an SSL certificate doesn’t mean your site is default secure.
  • You need to force https
  • Secure wordpress admin page
  • lock down your site before launching

Utilizing SEO Best Practices

Eliminating duplicate content so your site doesn’t get penalized

  • Search engines won’t list duplicate content - page title, page description
  • 100’s of duplicate pages can trigger Google’s spam detection and get you penalized.
  • Write clear and unique page titles - don’t reuse them
  • Don’t copy text descriptions from another site, it could be marked as duplicate.
  • Canonical tag
  • A tag that tells a search engine where the page content originally came from
  • <link rel=”canonical” href=”http://someweppage.com”>

Avoiding unnatural keyword usage

  • Write text on your site in a natural way
  • if you insert keywords into your text awkwardly, the page could be labeled spam
  • Cloaking: hiding text in the webpage. Mostly invisible to the user but clear in the HTML

Using rich media to promote your SEO results

  • Rich media such as large images, video, audio - should be optimized for web (fast loading)
  • Google will rank your page based on how quickly it loads.
  • Compress all images
  • Videos
  • Provide a summary or some content about this video nearby on the page, so that the search engine can get an idea about the content of the video.
  • Offer social share links to encourage link building.
  • Hosting your videos on YouTube will ensure that all devices can view your content

Minimizing your website loading speeds

  • pagespeed-insights - google tool
  • Load speed tips:
  • Optimize images and video
  • Use a quality hosting service
  • minify code
  • use a content delivery network (CDN)
  • Minify code:
  • removes all unnecessary (for parsing) characters in a file.
  • Using minified code with CDN’s can greatly reduce page load time

Correlating your SEO efforts to sales or other quantitative measures

  • Measurable Goals:
  • Product purchases
  • Newsletter signups
  • Social interactions
  • Product reviews
  • Google analytics:
  • Free tool from google that you install in your page
  • Set, View and track goals.
  • You can use these tracking methods to view user conversions

Implementing On-Page Optimization

Aligning your domain name with your target audience

  • Choosing a domain name that matches search terms helps discoverability
  • Try to keep your domain name no longer than three words (dollar shave club)
  • If you are in a crowded market - try making up a memorable word to stand out
  • using hyphens is ok, but no hyphens is best.
  • Domain name tips:
  • Shorter is better!
  • use on hyphen max, and only if required

Designing your landing page to drive deeper engagement

  • Home/Landing Page
  • should be big, bold and provide ‘calls to action’
  • Use a call to action (CTA) to encourage deeper exploration
  • Use a single prominent heading to clarify the page
  • Ensure important content is “above the fold” -  if a user has to scroll down to view important content it will likely be missed.
  • Break up the content - in pages that have long scrolls
  • Style the content to match the attitude of the site - Apple is stylish - Microsoft is crisp

Optimizing your product pages via breadcrumbs, page titles, and social links

  • Product Pages:
  • use breadcrumbs to simplify product navigation
  • Breadcrumb: a type of secondary navigation scheme that reveals the user’s location in a website or web application. Comes from Hansel and Gretel - leaving breadcrumbs to find their way back home.
  • Add social sharing buttons to encourage link building.
  • Use consistent page titles
  • (Product Name - Product Category | Business Name)
  • Write unique product descriptions. Do Not copy other descriptions found on a manufacturer’s page.

Understanding how mobile usability affects SEO

  • SEO Tips for Mobile
  • Check for the mobile-friendly badge
  • Run the Google mobile-friendly testing tool.
  • Avoid the use of FLash or data-heavy media on mobile
  • Consider the usability in your site design (fonts, buttons, etc)
  • Mobile-Friendly testing tool
  • run your site through the tool
  • the tool will test to see if your site uses responsive design
  • Mobile tips
  • The average mobile font size is 17px
  • keep headings short

  • Which Browser are they using

Employing heat mapping tools to measure and monitor page usage

  • Heat Mapping: A heat map for a website help you to get an understanding of how your visitors engage with your website. A visualization for where users are clicking.
  • Scroll Map: show how far down the user is scrolling - we want to place ways to sell or get subscribers in the hottest portion of your site.
