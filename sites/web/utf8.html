Unicode Transformation Format - 8 bit

UTF-8

UTF-8 is the most popular character encoding in use on the web today. With the rise of UTF-8 many encoding problems
that occured between systems running unique encodings to a particular
language or region have largely been elimenated.

Pretty interesting.I had heard of Unicode from using Python but never really looked too much into it. I didn't know that UTF-8  along with UTF-16, UTF-32 etc, was an encoding of Unicode. That's what the "U" in UTF-* stands for, "Unicode Transformation Format, 8-bit". The is an interesting blog post about Unicode and JavaScript over at the blog of Mathias Bynens

Some Definitions of words that I needed

  • Grapheme: characters or pieces of characters that you might write on a page to make words, sentences, or other pieces of text
  • Code Point: A number that maps to a particular abstract character. Code points make it so that we have a number pointing to a character without worrying about implementation details of how those numbers are stored for the computer to read. Encodings define how the code points map to particular sequence of bytes on disk and in memory.
